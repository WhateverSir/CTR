## 1. 简介
CTR 是支持tensorflow/pytorch/onnx多种框架的点击率预测项目，涵盖了DNN/LR/FM/DeepFM/DCN等模型。

## 2. 特性
* 支持tensorflow/pytorch/onnx多种框架
* 支持DNN/LR/FM/DeepFM/DCN等模型

## 3. 准备模型与数据
您下载本项目即可，包含多个模型，以及Movielens1M数据集。

## 4. 模型训练
进入demo文件夹，运行各个demo文件即可。
